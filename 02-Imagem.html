Colocando Imagem

Tag: img src=""

Voce pode colocar uma imagem como no essa embaixo utilizando a tag ( Img src="" )


almentar o tamanho ou diminuir e so colocar ( width="" height="" ) no img vai ficar assim>

    {
     <img src="file" alt="name" width="px" height="px">
    }
